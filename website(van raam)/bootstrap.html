Waarom heb ik dit bedrijf gekozen?

Ik heb dit bedijf gekozen omdat het een heel groot bedrijf is en bekend.
waardoor je veel keuze heb uit plaatjes producten en de layout van de website.
Ook heb ik het gekozen omdat ik zelf een fiets krijg die aangepast op mij maat en op de lengte.

Wat heb ik gemaakt

Ik moest voor deze opdracht een website voor een bedrijf van keuze.
Voor deze website moest ik een design maken in de design moest ik de vormgeving laten zien.
Ook moesten we voor deze website een database maken met gegevens die je moet
kunnen opvragen en laten zien in je Website.
